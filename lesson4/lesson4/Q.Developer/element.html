擬似要素とは

セレクターに付加するキーワードで、要素の「特定の部分」にスタイル付けができる。
擬似要素は擬似クラスと区別するためにcss３から二重コロン（::）を使用。
コロン（：）でも機能するものあるが「幅広いブラウザに対応させたい」等、特別な理由がない限り二重コロンを使う事。
擬似要素一例は以下

要素名 概要
::after 指定した要素の「直後」に擬似要素を作成
::before 指定した要素の「直前」に擬似要素を作成

下のDEMOでは★と♡を入れてるけど、floatを解除したい位置に空の擬似要素を入れて、clear:both;をかけるのがclearfixだね。
装飾をするのにもよく使うし、使う機会は多いかも。

HTML

				
				<p>これはテストです<p>

CSS

				
				p::befor{
				content: "★";
				}
				p::after{
				content: "♡";
				}

結果

これはテストです

要素名 概要
::first-line ブロック要素の最初の「行」にスタイルを適用。
行の為、要素の幅や文章の長さに左右されることに注意。

あんまり使った事ないけれど、<br>と組み合わせれば色々な使い方ができそう。

HTML

				
				<p>ブラウザの幅を狭めてみてください…省略…これはテストですブラウザの幅を狭めてみてください<p>

CSS

				
				p::firstLine{
				color: pink;
				}

擬似クラスとは

セレクターに付加するキーワードで、選択された要素に対して「特定の状態」時にスタイル付けができる。
擬似クラスには.classを扱う事はできない。
擬似クラスの一例は以下

要素名 概要
:hover 指定した要素の上にカーソルが乗った際のスタイルを指定

メニューや<a>にちょっとしたUIをつけたい時によく使うね。

HTML

					
					<p>これはテストです<p>

CSS

					
					p:hover{
					color: pink;
					}

結果

これはテストです

要素名 概要
:nth-child(n) 「n番目のE要素」にスタイルを指定する際に使います。

nの部分には整数・奇数(odd)偶数(even)数式を指定できます。
テーブルやリストの装飾に便利ですね。
数式というと難しく感じるかもしれませんが奇数の時この色！とかここからここまではこうする！等様々な指定ができます。
数式は調べるといろいろ出てきますヨ。

					li:nth-child(2n){
